💡 Petunjuk:
  • 1. Tambah kelas (misal: X TOI 1, XI TOI 2, ...) — data tersimpan di Firebase.
  • 2. Download template CSV, isi dengan data siswa (kolom no dan nama).
  • 3. Pilih kelas yang dituju, lalu unggah file CSV. Sistem akan mengganti semua data siswa di kelas tersebut dengan data baru.
  • 4. Anda juga bisa menghapus siswa satu per satu menggunakan tombol "Hapus".
⚡ Data disimpan di Firebase Firestore. Pastikan koneksi internet aktif.
